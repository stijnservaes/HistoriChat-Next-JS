# HistoriChat-Next-JS
Frontend application for HistoriChat App. Backend on https://github.com/stijnservaes/HistoriChat.

## Status
### Finished



## Getting Started
### Install Instruction
1. Clone the repo
2. Install NPM packages and build
  ```sh
  npm install && next build
  ```
3. Start server
  ```
  npm start
  ```
